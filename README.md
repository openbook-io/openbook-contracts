# OpenBook Contracts (ERC1400)

### Prerequisites

- Node.js > 10.1

- Truffle > 5.1.12 

- Ganache (Local Geth Node) > 2.1.2


### Run Project

- Install Dependencies
```
$ npm i
```

- Compile contracts
```
$ npm run truffle:compile
```

- Migrate Contracts
```
$ npm run truffle:migrate
```

- Unit Testing
```
$ npm run truffle:test
```
